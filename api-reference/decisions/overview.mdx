## How We Make Decisions

```mermaid
flowchart LR
    A["Problem\nIdentified"] --> B["List\nOptions"]
    B --> C["Evaluate\nTrade-offs"]
    C --> D["Team\nAlignment"]
    D --> E["Write ADR\n(Context + Decision\n+ Consequences)"]
    E --> F["Implement"]
    F --> G["Review\n(if outcome differs)"]

```
